About

This application provides simple, accessible medical assistance guidance for use in emergency or wartime situations.

Details

This tool is designed for use in low-resource, high-pressure environments, offering touch-friendly navigation and offline accessibility. Content can be customised and expanded based on operational needs.

Credits

This project is supported by contributors focused on field medicine, humanitarian response, and mobile-first interface design. Future versions will include multilingual support and integration with field medical protocols.
